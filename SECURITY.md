# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 3.0.x   | ✅                 |
| 2.5.x   | ✅                 |
| < 2.4.0 | :x:                |

## Reporting a Vulnerability

Please create a **Draft Security Advisory** from the [Security and quality](https://github.com/JS8Call-improved/JS8Call-improved/security/advisories)
page.

We will normally respond within 1-3 days. Please try to be as detailed as possible; ensure you include the OS(s) and the version you found the issue on.

Thank you!
